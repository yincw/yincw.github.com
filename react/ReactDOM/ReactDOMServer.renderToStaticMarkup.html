React API

ReactDOMServer.renderToStaticMarkup(element)

ReactDOMServer.renderToStaticMarkup 是第二个服务器端渲染函数。

除了不会包含 React 的 data 属性外，它和 ReactDOMServer.renderToString 没有区别。当且仅当你不打算在客户端渲染这个 React 组件时，才应该选择使用 ReactDOMServer.renderToStaticMarkup 函数。

ReactDOMServer.renderToStaticMarkup(<App />);
